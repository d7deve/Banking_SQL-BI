# Fraud Detection Project

## Tools Used
- SQL Server
- Power BI

## Description
Analyzed financial transactions to detect fraudulent patterns using SQL queries and Power BI dashboards.

## Key Insights
- Fraud concentrated in TRANSFER and CASH_OUT
- High-value transactions are riskier
- Balance inconsistencies indicate fraud
